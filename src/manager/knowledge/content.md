## 2️⃣ 콘텐츠 톤 가이드

### 일관된 브랜드 보이스

```
키워드 5개:
1. 객관적 (상품 안 권함)
2. 따뜻함 (오상열 CFP 인격)
3. 구체적 (수치 / 사례 / 단계)
4. 한국적 (현실 반영)
5. 행동 유도 (다음 1단계 명확)

피해야 할 표현:
❌ "대박" / "돈복사" (자극적)
❌ "100% 보장" (과장)
❌ "이번에만" (조급함 강요)
❌ "당신만 모르는" (음모론적)
❌ "강의장 분위기" (니즈 있는 사람만)

권장 표현:
✅ "25년 데이터" (객관성)
✅ "단계별" (체계성)
✅ "매일 1분" (실현 가능)
✅ "자녀 / 노후" (공감)
✅ "FIRE 가능" (구체 목표)
```

## 5️⃣ 데일리 퀴즈 콘텐츠

### 퀴즈 구조 (30초 형식)

```
앱 진입 시:
┌──────────────────────────┐
│ 💎 오늘의 퀴즈              │
│                            │
│ Q. 비상예비자금의           │
│    적정 규모는?              │
│                            │
│ ① 생활비 1개월              │
│ ② 생활비 3-6개월            │
│ ③ 생활비 1년                │
│ ④ 생활비 무관               │
│                            │
│ [긁어서 정답 확인]          │
└──────────────────────────┘

긁기 (스크래치) 효과:
- 사용자 손가락으로 카드 긁음
- 정답 / 오답 즉시 공개
- 정답 시: "정답 ✅ +1코인"
- 오답 시: "오답 ❌ 정답: ② / 해설"

코인 적립:
- 정답 1개 = 1코인
- 30일 누적 = 약 30코인
- 100코인 = 1,000원 결제 할인
```

### 퀴즈 카테고리 (라이트한 주제)

```
1. 재테크 상식 (50%)
   - 4% Rule
   - 복리 / 단리 차이
   - 인플레이션 영향

2. DESIRE 단계 (25%)
   - 각 단계 의미
   - 한국 적용
   - 사례 판단

3. 시사 / 경제 (15%)
   - 최근 뉴스 (가벼운)
   - 정책 변화
   - 시장 흐름

4. 행동경제학 (10%)
   - 손실 회피
   - 군중 심리
   - 매몰비용
```

### 퀴즈 자동 생성 (제니야)

```
대표님 콘텐츠 자산 활용:
- 유튜브 영상 (47K 채널)
- 365 팟캐스트 에피소드
- 강의 PDF / 슬라이드

자동 생성 흐름:
1. 영상 자막 / 강의 자료 추출
2. Claude API로 퀴즈 1개 생성
   - 4지선다
   - 정답 + 해설
   - 난이도 (쉬움/중간/어려움)
3. 대표님 검토 (주 1회)
4. 승인 시 큐 적재
5. 매일 자동 발송
```
