## 2️⃣ DESIRE 6단계 (핵심 방법론) - 상세

### 전체 개요

```
DESIRE = 재무 자유로 가는 6단계 로드맵
순서가 절대적: 건너뛰면 무너진다
데이브 램지 BabyStep과 한국 현실 결합
최종 목표: FIRE (Financial Independence Retire Early)
```

### D - Debt (신용대출 청산) ⭐ 가장 시급

```
정의: 고금리 신용대출, 카드론, 마이너스통장 등 모두 상환

왜 1번?
- 고금리 부채 = 마이너스 수익 (15-20%)
- 부채 갚기 = 가장 확실한 투자 수익
- 부채 있으면 다음 단계 무의미

세부 단계:
- D1: 가장 고금리 1순위 청산
- D2: 두 번째 고금리 청산
- D3: 모든 신용대출 0원 (Debt Free)

기간 목표: 6개월 ~ 2년

핵심 전략:
- 눈덩이 방식 (작은 부채부터) - 심리적 동기
- 또는 눈사태 방식 (고금리부터) - 수학적 효율
- 가용 현금 흐름 100% 부채 상환에 투입
- 비상예비자금은 최소 1개월치만 유지

위험 신호:
- 신규 신용대출 받음 → 단계 후퇴
- 카드 할부 / 리볼빙 → 부채로 간주
```

### E - Emergency (비상예비자금) ⭐ 안전망

```
정의: 생활비 3-6개월치 즉시 인출 가능 자금

왜 2번?
- 부채 청산 후 첫 안전망
- 갑작스런 실직 / 병원비 / 사고 대비
- 이게 없으면 다음 단계 가다가 부채로 회귀

목표 금액:
- 1차 목표: 생활비 3개월치
- 최종 목표: 생활비 6개월치
- 자녀 있는 가정: 9-12개월치 권장

저장 위치 (중요):
- 즉시 인출 가능 (CMA, 파킹통장)
- 절대 주식/펀드 X (변동성)
- 절대 적금 X (해지 손실)
- 정기예금 6개월 OK

세부 단계:
- E1: 1개월치 (긴급용)
- E2: 3개월치 (안전망)
- E3: 6개월치 (Emergency Free)

핵심 전략:
- 매월 저축액의 100% E단계 집중
- D 단계 끝나자마자 즉시 시작
- 절대 손대지 않는 영역 (라벨링)
```

### S - Saving (적립식 저축) - 종잣돈

```
정의: 매월 정기적으로 종잣돈 만들기

왜 3번?
- 부채 0 + 비상자금 OK 후 본격 자산 증식
- 종잣돈 1억은 인생의 분기점
- 복리 효과의 출발점

목표 금액:
- 1차: 1,000만원
- 2차: 5,000만원  
- 최종: 1억원

저장 방식:
- 청약저축 (필수)
- IRP / 연금저축 (세제 혜택)
- ISA (비과세)
- 적립식 펀드 (인덱스 추천)

세부 단계:
- S1: 1,000만원
- S2: 5,000만원
- S3: 1억원 달성

핵심 전략:
- 자동이체 (의지력 X, 시스템 O)
- 월급의 30% 이상 저축 (고소득자 50%)
- 절대 중도 인출 X
- 세금 혜택 상품 우선

기간: 보통 5-7년
```

### I - Investment (투자자산 / 10억 목돈)

```
정의: 본격적 자산 증식 단계

왜 4번?
- 종잣돈 만든 후 진짜 부 축적
- 단순 저축으로는 인플레이션 못 따라잡음
- 10억 = 한국 사회 부자의 실질적 기준

목표 금액:
- 1차: 3억원 (중산층 진입)
- 2차: 5억원 (상위 중산층)
- 최종: 10억원 (실질적 부자 진입)

투자 영역:
- 주식 (인덱스 + 우량주)
- 부동산 (주거 외 투자용)
- 펀드 / ETF
- 채권 (안정성)
- 대안 투자 (ESG / 테마)

세부 단계:
- I1: 3억원
- I2: 5억원
- I3: 10억원 (Investment Free)

핵심 전략:
- 분산 투자 (주식 60% / 부동산 30% / 현금 10%)
- 장기 보유 (10년 이상)
- 시장 타이밍 X, 시간 분산 O
- 매년 포트폴리오 리밸런싱

기간: 10-20년
```

### R - Real Estate (담보대출 / 부동산)

```
정의: 주거 안정 + 부동산 레버리지 활용

왜 5번?
- 자산 10억 도달 후 부동산 본격 진입
- 한국에서 부동산 = 부의 핵심
- 담보대출 = 좋은 빚 (생산적 부채)

세부 단계:
- R1: 자가 주택 마련 (담보대출 50% 이내)
- R2: 추가 투자 부동산
- R3: 부동산 포트폴리오 완성

핵심 전략:
- 자가 우선 (주거 안정)
- 입지 분석 (직주근접, 학군, 교통)
- 담보대출 활용 (LTV 70% 이내)
- 임대 수익 + 시세 차익 결합

위험 관리:
- 부동산 비중 50% 초과 X
- 담보대출 상환 능력 우선
- 시장 사이클 인지 (10년 주기)

기간: 5-10년
```

### E - Economic Freedom (경제적 자유 / FIRE) ⭐ 최종 목표

```
정의: 일하지 않아도 생활비가 나오는 상태

왜 마지막?
- 25년 재무 여정의 종착지
- 진정한 부자 = 시간의 자유
- 자녀 / 손자에게 물려줄 자산 시작

목표:
- 자산: 30억원 이상 (소비 수준에 따라)
- 패시브 인컴: 월 생활비의 150%
- 자산 운용 시스템 자동화

세부 단계:
- E1: 조기 은퇴 가능 (50대 초)
- E2: 진정한 FIRE (자산 100% 패시브)
- E3: 다음 세대 자산 이전 시작

핵심 전략:
- 4% Rule (자산의 4%로 생활)
- 배당주 / 임대수익 / 채권 이자
- 인플레이션 헤지
- 상속 계획 (세금 최적화)

특징:
- 강제 일이 아닌 즐거운 일만
- 시간 = 가장 값비싼 자산
- 가족 / 건강 / 의미 있는 활동
```

### DESIRE 진단 시 주의 사항

```
1. 순서 절대 준수
   - 자산 많아도 부채 있으면 D
   - 부동산 보유해도 비상자금 없으면 E

2. 단계 후퇴 가능
   - 신용대출 받으면 → D로 회귀
   - 비상자금 소진 → E로 회귀

3. 한국 특수성
   - 전세 / 월세 vs 자가 (부동산)
   - 연금저축 / IRP (세제)
   - 청약 (정부 지원)

4. 나이별 권장
   - 20대: D-E-S 집중
   - 30대: S-I 본격화
   - 40대: I-R 확장
   - 50대+: R-E 마무리
```

## 3️⃣ 금융집짓기® 8대 영역 (특허)

### 전체 구조

```
금융집짓기 = 재무를 "집"에 비유한 종합 진단 체계
DESIRE가 "여정"이라면, 금융집짓기는 "건축물"
8대 영역 = 집의 8개 핵심 부분
```

### 1. 노후 (Retirement) - 지붕

```
역할: 비를 막는 지붕 = 노후의 기둥

핵심 항목:
- 국민연금 (예상 수령액)
- 퇴직연금 (DC/DB)
- 개인연금 (IRP, 연금저축)
- 노후 자금 목표 (월 OO만원)

진단 질문:
- 65세에 매월 얼마 받을 수 있나?
- 부족분은 얼마인가?
- 부족분 충당 계획 있나?

위험 신호:
- 국민연금만 의존
- 개인연금 0원
- 노후 자금 목표 없음
```

### 2. 부채 (Debt) - 기초

```
역할: 무너지면 안 되는 기초

핵심 항목:
- 신용대출 (잔액, 금리, 만기)
- 담보대출 (LTV, DTI)
- 카드론, 리볼빙
- 학자금 / 사업자대출

진단 질문:
- 부채 비율 (자산 대비)?
- 월 상환 부담 (소득 대비)?
- 고금리 부채 우선 청산 계획?

황금 비율:
- 부채/자산: 30% 이내
- 월 상환/소득: 30% 이내
```

### 3. 저축 (Saving) - 내력벽

```
역할: 집을 지탱하는 내력벽

핵심 항목:
- 청약저축
- 적금 / 정기예금
- ISA / IRP
- 연금저축

진단 질문:
- 월 저축률 (소득 대비)?
- 비상예비자금 충분한가?
- 세제 혜택 활용하나?

권장 비율:
- 월 저축: 소득의 30% (직장인) / 50% (고소득)
- 비상자금: 생활비 3-6개월치
```

### 4. 투자 (Investment) - 인테리어

```
역할: 집을 풍요롭게 하는 인테리어

핵심 항목:
- 주식 (개별 / 펀드 / ETF)
- 채권
- 대안 투자 (P2P, 크라우드펀딩)
- 외화 자산

진단 질문:
- 투자 자산 비율?
- 분산 투자 되어있나?
- 위험 감수 수준 적절한가?

원칙:
- 분산 (주식/채권/현금)
- 장기 (10년 이상)
- 시간 분산 (매월 적립)
```

### 5. 세금 (Tax) - 누수 방지

```
역할: 새는 물 막는 누수 방지

핵심 항목:
- 소득세 (근로 / 사업)
- 양도세 (부동산 / 주식)
- 상속세 / 증여세
- 종합부동산세

진단 질문:
- 세제 혜택 상품 가입했나?
- 절세 전략 있나?
- 가족 합산 최적화?

전략:
- 연말정산 활용 (IRP, 연금저축)
- 양도세 비과세 활용
- 증여 시기 분산
- 사업자 세금 관리
```

### 6. 부동산 (Real Estate) - 외벽

```
역할: 집의 외벽 = 가장 큰 자산

핵심 항목:
- 자가 주택
- 투자 부동산
- 토지 / 상가
- 임대 수익

진단 질문:
- 자가 vs 임대?
- 부동산 비중 적절한가?
- 임대 수익률은?

황금 비율:
- 부동산/총자산: 50% 이내
- LTV: 70% 이내
- 임대수익률: 4% 이상
```

### 7. 보험 (Insurance) - 보안 시스템

```
역할: 재난 대비 보안 시스템

핵심 항목:
- 종신보험 / 정기보험 (사망)
- 실손 의료보험 (치료)
- 중대 질환 보험 (암/뇌/심장)
- 운전자 / 화재 / 배상책임

진단 질문:
- 사망보장 충분한가?
- 의료비 100% 커버되나?
- 중복 / 누락 가입 없나?
- 보험료 부담 적정한가?

권장:
- 보험료: 소득의 8% 이내
- 사망보장: 연소득 5-10배
- 실손 1개 (필수)
- 중대질환 1개
- 종신/정기 1개

⭐⭐ 전문가 플랜 핵심 영역
```

### 8. 상속/증여 (Legacy) - 정원

```
역할: 다음 세대로 이어지는 정원

핵심 항목:
- 유언 / 신탁
- 증여 계획 (자녀, 손주)
- 보험금 수익자
- 사업 승계

진단 질문:
- 유언장 작성했나?
- 증여 시기 / 금액 계획?
- 가족 합의 됐나?
- 세금 최적화 됐나?

전략:
- 10년 단위 증여 (한도 활용)
- 보험금 활용 (상속세 재원)
- 사업 가업 승계 (특례)
- 손주 직접 증여 (세대 건너뛰기)
```

### 8대 영역 진단 점수 산출

```
각 영역 100점 만점 평가
8개 영역 평균 = 종합 재무 건강 점수

예시:
노후: 85점
부채: 60점 (위험)
저축: 75점
투자: 70점
세금: 50점 (위험)
부동산: 80점
보험: 65점
상속: 30점 (매우 위험)
─────────────
종합: 64.4점

→ 부채 / 세금 / 상속 우선 개선 권장
```

## 4️⃣ AI머니야 사업 모델

### 핵심 차별화

```
일반 재무 앱 (토스, 뱅크샐러드):
- 자동 데이터 연동 (편리)
- 상품 판매 위주 (수익)
- AI 챗봇 (얕음)

AI머니야:
- 25년 CFP 노하우 (깊이)
- 상품 판매 0 (객관성)
```
